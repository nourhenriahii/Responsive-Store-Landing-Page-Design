# 💄MAC — Beauty Redefined | Responsive Store Landing Page

A fully responsive landing page for a makeup/cosmetics online store, built with pure HTML5 and CSS3 (no frameworks).

## 📌 Project Overview

This project was built as part of the **Generative AI for Professionals** training program.
The goal was to design a responsive Landing Page for an online store, including:

- A header with a navigation menu
- A hero/home section
- A category showcase
- A bestsellers / product section
- A promo banner
- Customer testimonials section
- A store locator section
- A newsletter signup
- A footer

## 🖼️ Important Note About Images

> ⚠️ **All images used in this project (product photos, banners, testimonial photos, store visuals, logo, etc.) were generated using AI tools.** They are used purely for design/demo purposes and do not represent real products, real customers, or real stores.

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Font Awesome (icons)
- Google Fonts (Cormorant Garamond + Jost)
- Media Queries for responsiveness

## ✨ Features

- Sticky/dark-themed header with logo, navigation links, and icon shortcuts (search, account, cart)
- Hero section with background image and call-to-action buttons
- "Shop by Category" section with hover image effects
- Bestsellers section with 4 product cards (image, name, description, price, "add to bag" button, badges like "NEW" / "BEST SELLER")
- Promo banner section for limited edition collections
- Inspiration/Testimonials section with customer reviews and star ratings
- Store locator section with a list of store locations and an image
- Newsletter subscription form
- Multi-column footer with brand info, shop links, help links, about links, and social icons
- Fully responsive layout using CSS Grid/Flexbox and media queries (desktop, tablet, mobile)
